# Enneagram — Type Tuning

The Enneagram is a personality system organized around nine core types, each with a different core motivation, fear, and adaptive strategy. Unlike MBTI's four-letter style, Enneagram types are numbered 1–9.

## Find your type

Free tests:
- [Eclectic Energies](https://www.eclecticenergies.com/enneagram/test) — free, no signup, multiple test formats
- [Truity Enneagram Test](https://www.truity.com/test/enneagram-personality-test) — short, free
- [Crystal Knows](https://www.crystalknows.com/enneagram-test) — free with signup

Your dominant type is the number you score highest on. **Wings** (the adjacent number that influences your dominant) aren't addressed in v1 — start with your dominant.

## MBTI vs Enneagram tuning

MBTI tuning focuses on **communication style** — terseness, energy, structure. Enneagram tuning focuses on **core motivation** — what the user is protecting and what they're seeking. They complement rather than overlap.

You can layer them. Paste both your MBTI tuning and your Enneagram tuning into the agent's system prompt. The agent will calibrate on both dimensions.
